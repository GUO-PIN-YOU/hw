輸入多項式：
每次新增項需要 O(1)。
若陣列需要擴展，最壞情況是 O(n)，其中 n 是項目數量。
整體平均時間複雜度為 O(n)。
輸出多項式：
逐一列印多項式的所有項，時間複雜度為 O(n)。
空間複雜度 (Space Complexity)
使用動態陣列儲存非零項，空間需求為 O(n)，其中 n 是項目數量。
額外的空間需求為常數，因此總空間複雜度為 O(n)。
